# 校园 Moments - 前端完整接口文档

## 📋 基本信息

- **接口地址**: `http://localhost:8080` (开发环境)
- **认证方式**: JWT Bearer Token
- **数据格式**: JSON
- **字符编码**: UTF-8

---

## 🔐 认证说明

### 获取Token后需要在需要认证的接口请求头中携带：

```http
Authorization: Bearer <你的token>
```

## 🏠 用户认证模块

### 1. 用户注册
**用途**: 新用户账号注册

```http
POST /auth/register
```

**请求参数**:
```json
{
  "username": "张三",         // 用户名，3-20个字符
  "phone": "13800138000",    // 手机号，中国大陆格式
  "password": "Password123",  // 密码，6-20位
  "code": "123456"          // 验证码（开发环境可传任意6位数字）
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "userId": "0000000001",
    "username": "张三",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
```

### 2. 用户登录
**用途**: 用户账号登录

```http
POST /auth/login
```

**请求参数**:
```json
{
  "account": "13800138000",  // 手机号或用户名
  "password": "Password123"  // 密码
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }
}
```

### 3. 发送验证码
**用途**: 获取手机验证码

```http
POST /auth/send-verification
```

**请求参数**:
```json
{
  "phone": "13800138000"  // 手机号
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "验证码发送成功",
  "data": null
}
```

### 4. 重置密码
**用途**: 通过验证码重置密码

```http
POST /auth/verify-and-reset
```

**请求参数**:
```json
{
  "phone": "13800138000",   // 手机号
  "code": "123456",        // 验证码
  "newPassword": "New123456" // 新密码
}
```

### 5. 退出登录
**用途**: 用户退出登录

```http
POST /api/auth/logout
```

**请求头**:
```http
Authorization: Bearer <token>
```

## 👤 用户管理模块

### 1. 获取用户信息
**用途**: 获取当前登录用户的详细信息

```http
GET /api/users/profile
```

**请求头**:
```http
Authorization: Bearer <token>
```

```json
{

  "message": "获取成功",
  "data": {

    "username": "张三",
    "phone": "13800138000",

  }

```

**用途**: 更新用户个人资料

```http
PUT /api/users/profile
```

**请求参数**:
```json
{
  "nickname": "新昵称",

  "gender": 1,              // 0:未知,1:男,2:女
```

```http
PUT /api/users/password
```

**请求参数**:
```json

}
```

## 📝 动态内容模块

### 1. 获取动态列表
**用途**: 获取首页动态内容流

```http
GET /moments
```

**查询参数**:
- `page`: 页码（默认1）
- `pageSize`: 每页数量（默认10）

**响应示例**:
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "moments": [
      {
        "id": 8,
        "content": "这是我的动态内容",
        "authorId": "0000000001",
        "tags": ["生活", "分享"],
        "media": [
          {
            "type": "image",
            "url": "https://oss.example.com/image1.jpg",
            "thumbnail": "https://oss.example.com/thumb1.jpg"
          }
        ],
        "visibility": 1,           // 1:公开,2:好友,3:私密
        "likeCount": 10,
        "commentCount": 5,
        "status": 1,
        "createdAt": "2023-01-01T00:00:00Z",
        "author": {
          "userId": "0000000001",
          "username": "张三",
          "avatar": "https://oss.example.com/avatar.jpg"
        }
      }
    ],
    "pagination": {
      "page": 1,
      "pageSize": 10,
      "total": 100
    }
  }
}
```

### 2. 发布动态
**用途**: 创建新的动态内容

```http
POST /api/moments
```

**请求参数**:
```json
{
  "content": "这是我要发布的内容",
  "visibility": 1,           // 1:公开,2:好友,3:私密
  "status": 1,               // 1:正常,2:删除
  "tags": ["生活", "分享"],
  "media": [
    {
      "type": "image",
      "url": "https://oss.example.com/image1.jpg"
    }
  ]
}
```

### 3. 获取动态详情
**用途**: 获取单条动态的详细信息

```http
GET /api/moments/{id}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "id": 8,
    "content": "这是我的动态内容",
    "authorId": "0000000001",
    "tags": ["生活", "分享"],
    "media": [...],
    "visibility": 1,
    "likeCount": 10,
    "commentCount": 5,
    "status": 1,
    "createdAt": "2023-01-01T00:00:00Z",
    "author": {...},
    "comments": [...]
  }
}
```

### 4. 获取我的动态
**用途**: 获取当前用户发布的动态列表

```http
GET /api/moments/my?page=1&pageSize=10
```

### 5. 删除动态
**用途**: 删除自己发布的动态

```http
DELETE /api/moments/{id}
```

### 6. 修改动态
**用途**: 修改已发布的动态内容

```http
PATCH /api/moments/{id}
```

**请求参数**:
```json
{
  "content": "修改后的内容",
  "visibility": 1,
  "tags": ["新标签"]
}
```

## 📊 响应格式说明

### 成功响应
```json
{
  "code": 200,
  "message": "操作成功",
  "data": { ... }
}
```

### 错误响应
```json
{
  "code": 400,             // 错误码
  "message": "错误描述",     // 错误信息
  "data": null
}
```

### 常见错误码
- `200`: 成功
- `400`: 请求参数错误
- `401`: 未认证或token无效
- `403`: 权限不足
- `404`: 资源不存在
- `500`: 服务器内部错误

## 📱 前端使用示例

### 登录获取Token
```javascript
const response = await fetch('http://localhost:8080/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    account: '13800138000',
    password: 'Password123'
  })
});

const data = await response.json();
const token = data.data.token;
```

### 发表评论
```javascript
const response = await fetch('http://localhost:8080/api/comments', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    momentId: 8,
    content: '这是我的评论'
  })
});
```

### 点赞动态
```javascript
const response = await fetch('http://localhost:8080/api/likes', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    targetId: 8,
    targetType: 1
  })
});
```

## 🎯 开发注意事项

1. **Token管理**: Token有效期为24小时，过期需要重新登录
2. **文件上传**: 支持常见图片格式(jpg,png,gif)和视频格式
3. **分页查询**: 所有列表接口都支持分页，建议pageSize不超过50
4. **权限控制**: 删除操作只能操作自己的内容，管理员可操作所有内容
5. **数据验证**: 所有接口都会进行参数验证，请按规范传递数据
6. **错误处理**: 请根据code和message进行错误提示，不要直接展示data内容
